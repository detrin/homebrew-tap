# Homebrew Tap

Homebrew formulae for detrin's tools.

## Usage

```bash
brew tap detrin/tap
brew install brow

# After installation, install Chromium:
playwright install chromium
```

## Available Formulae

- **brow** - Standalone Playwright CLI for AI agent browser automation ([repo](https://github.com/detrin/brow))

## Upgrade

```bash
brew update
brew upgrade brow
```
